## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Neon-Carrot/neoncarrot.com.au.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally
   ```sh
   npm run dev
   ```
